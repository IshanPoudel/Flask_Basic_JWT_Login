# TradeMinds

## Frontend
The frontend for TradeMinds can be found [here](https://github.com/AnshSharma2001/stock-prediction).

## Backend

### Computational Layer
The computational layer is where data analysis and model execution take place. This includes subsystems for:
- Ranking machine learning models
- Standardizing incoming data
- Predicting market trends

### Backend Database Layer
The backend database layer ensures the secure, compliant, and efficient handling of all data within TradeMinds. It includes specialized subsystems for:
- Managing user data
- Managing model data
- Standardizing data from external APIs like Alpha Vantage.
